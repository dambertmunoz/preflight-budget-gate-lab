# LinkedIn Architecture Note

Hook: architecture is not about adding more layers; it is about making the risky decisions explicit.

Topic: Tu agente gasta antes de pensar

Angle: El anti-patrón junior es medir tokens al final y llamar "control" a una autopsia de costo. En agentes con tools y side effects, el budget governor debe decidir sobre la intención antes del primer call; si no, no gobiernas riesgo ni gasto, solo registras el incidente.
